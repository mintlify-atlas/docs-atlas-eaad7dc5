## What is Start UI [web]?

Start UI [web] is an opinionated frontend starter repository created and maintained by the [BearStudio Team](https://www.bearstudio.fr/team). It represents our team's up-to-date stack that we use when creating web apps for our clients.

Built with modern technologies like TanStack Start, TypeScript, Prisma, and Better Auth, this starter provides everything you need to build production-ready full-stack applications.

## Quick Start

Get started with Start UI [web] in minutes:

<CodeGroup>
```bash npm
pnpm create start-ui -t web myApp
```

```bash Manual Setup
git clone https://github.com/bearstudio/start-ui-web.git myApp
cd myApp
pnpm install
cp .env.example .env
pnpm dk:init
pnpm db:init
pnpm dev
```
</CodeGroup>

## Technology Stack

Start UI [web] is built on a carefully selected stack of modern technologies:

- **[Node.js](https://nodejs.org)** ≥ 22 - JavaScript runtime
- **[TypeScript](https://www.typescriptlang.org/)** - Type-safe JavaScript
- **[React](https://react.dev/)** 19 - UI framework
- **[TanStack Start](https://tanstack.com/start)** - React meta-framework
- **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first CSS
- **[shadcn/ui](https://ui.shadcn.com/)** - Component library
- **[React Hook Form](https://react-hook-form.com/)** - Form management
- **[oRPC](https://orpc.unnoq.com/)** - Type-safe RPC framework
- **[Prisma](https://www.prisma.io/)** - Database ORM
- **[Better Auth](https://www.better-auth.com/)** - Authentication
- **[Storybook](https://storybook.js.org/)** - Component development
- **[Vitest](https://vitest.dev/)** - Unit testing
- **[Playwright](https://playwright.dev/)** - E2E testing
