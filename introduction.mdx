# Introduction to Start UI [web]

Start UI [web] is an opinionated, production-ready frontend starter repository created and maintained by the [BearStudio Team](https://www.bearstudio.fr/team). It represents an up-to-date stack for building modern web applications with a focus on developer experience, type safety, and best practices.

## What is Start UI [web]?

Start UI [web] is a comprehensive full-stack starter that provides everything you need to build a modern web application:

- **Full-stack React framework** powered by TanStack Start
- **End-to-end type safety** with TypeScript throughout
- **Modern authentication** using Better Auth with multiple providers
- **Database integration** with Prisma ORM and PostgreSQL
- **Beautiful UI components** from shadcn/ui with Tailwind CSS
- **API layer** with oRPC for type-safe client-server communication
- **Testing suite** including Vitest for unit tests and Playwright for E2E tests
- **Developer tools** including Storybook for component development

## Who is it for?

Start UI [web] is perfect for:

- **Development teams** looking for a solid foundation with established patterns
- **Startups** needing to move fast with a production-ready stack
- **Agencies** building web applications for clients with consistent architecture
- **Developers** wanting to learn modern web development practices
- **Projects** requiring authentication, database integration, and type safety out of the box

## Key Technologies

Start UI [web] leverages a carefully selected set of modern technologies:

### Core Framework
- **[TanStack Start](https://tanstack.com/start)** - Full-stack React framework with file-based routing
- **[React 19](https://react.dev/)** - Latest version with improved performance and features
- **[TypeScript](https://www.typescriptlang.org/)** - Type safety across the entire application
- **[Node.js ≥22](https://nodejs.org)** - Modern JavaScript runtime

### Styling & UI
- **[Tailwind CSS 4](https://tailwindcss.com/)** - Utility-first CSS framework
- **[shadcn/ui](https://ui.shadcn.com/)** - High-quality, accessible React components
- **[Lucide React](https://lucide.dev/)** - Beautiful icon library

### Backend & Database
- **[Prisma](https://www.prisma.io/)** - Type-safe ORM for database operations
- **[PostgreSQL](https://www.postgresql.org/)** - Robust relational database
- **[Better Auth](https://www.better-auth.com/)** - Modern authentication solution
- **[oRPC](https://orpc.unnoq.com/)** - Type-safe API layer

### Forms & Validation
- **[React Hook Form](https://react-hook-form.com/)** - Performant form management
- **[Zod](https://zod.dev/)** - Schema validation with TypeScript inference

### Testing & Quality
- **[Vitest](https://vitest.dev/)** - Fast unit and component testing
- **[Playwright](https://playwright.dev/)** - Reliable end-to-end testing
- **[Storybook](https://storybook.js.org/)** - Component development and documentation

### Development Tools
- **[pnpm](https://pnpm.io/)** - Fast, disk space efficient package manager
- **[Docker](https://www.docker.com/)** - Containerized development environment
- **[Vite](https://vitejs.dev/)** - Lightning-fast build tool

## What's Included?

The starter comes with several pre-built features to accelerate your development:

- **Authentication flows** - Login, registration, email verification, password reset
- **User management** - Admin panel for managing users, roles, and permissions
- **Account settings** - Profile management, avatar uploads, preferences
- **Example CRUD operations** - Books, genres, authors, and publishers
- **File uploads** - Integration with S3-compatible storage (MinIO)
- **Email system** - Transactional emails with React Email templates
- **Internationalization** - Multi-language support with i18next (English, French, Arabic, Swahili)
- **Dark mode** - System-aware theme switching
- **Responsive design** - Mobile-first, fully responsive layouts
