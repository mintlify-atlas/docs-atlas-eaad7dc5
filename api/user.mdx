## Overview

The User API provides comprehensive admin-level endpoints for managing users and their sessions. All endpoints require authentication and specific admin permissions.

<Warning>
  These are **admin-only** endpoints. For user self-service operations, see the [Account API](/api/account).
</Warning>

## Permissions

User API endpoints require the following permissions:

- `user:list` - View users and their details
- `user:create` - Create new users
- `user:set-role` - Update user roles and information
- `user:delete` - Delete users
- `session:list` - View user sessions
- `session:revoke` - Revoke user sessions

```typescript
GET /users
```

#### Code Example

<CodeGroup>

```typescript React Query
import { orpc } from '@/lib/orpc/client';

function UserList() {
  const { data, isLoading, fetchNextPage, hasNextPage } = 
    orpc.user.getAll.useInfiniteQuery({
      input: (page) => ({
        searchTerm: '',
        limit: 20,
        cursor: page?.nextCursor,
      }),
      getNextPageParam: (lastPage) => 
        lastPage.nextCursor ? { nextCursor: lastPage.nextCursor } : undefined,
    });

  if (isLoading) return <div>Loading users...</div>;

  return (
    <div>
      <table>
        <thead>
          <tr>
            <th>Name</th>
            <th>Email</th>
            <th>Role</th>
            <th>Verified</th>
          </tr>
        </thead>
        <tbody>
          {data?.pages.flatMap(page => page.items).map(user => (
            <tr key={user.id}>
              <td>{user.name}</td>
              <td>{user.email}</td>
              <td>{user.role}</td>
              <td>{user.emailVerified ? '✓' : '✗'}</td>
            </tr>
          ))}
        </tbody>
      </table>
      
      {hasNextPage && (
        <button onClick={() => fetchNextPage()}>
          Load More
        </button>
      )}
      
      <p>Total: {data?.pages[0]?.total} users</p>
    </div>
  );
}
```

```typescript React Query
        Create User
      </button>
    </form>
  );
}
```

#### Code Example

<CodeGroup>

```typescript React Query
import { orpc } from '@/lib/orpc/client';

function DeleteUserButton({ userId, currentUserId }) {
  const { mutate, isPending } = orpc.user.deleteById.useMutation({
    onSuccess: () => {
      orpc.user.getAll.invalidate();
      router.navigate({ to: '/admin/users' });
    },
    onError: (error) => {
      if (error.message.includes('cannot delete yourself')) {
        alert('You cannot delete your own account');
      }
    },
  });

  const handleDelete = () => {
    if (userId === currentUserId) {
      alert('You cannot delete your own account');
      return;
    }
    
    if (confirm('Are you sure you want to delete this user?')) {
      mutate({ input: { id: userId } });
    }
  };

  return (
    <button 
      onClick={handleDelete} 
      disabled={isPending || userId === currentUserId}
    >
      {isPending ? 'Deleting...' : 'Delete User'}
    </button>
  );
}
```

### Revoke All User Sessions

Revoke all active sessions for a user (force logout from all devices).

```typescript
POST /users/{id}/sessions/revoke
```

**Source**: `src/server/routers/user.ts:287-325`

#### Authentication

<ParamField path="permissions" type="object">
  Requires `session:revoke` permission
</ParamField>

#### Path Parameters

<ParamField path="id" type="string" required>
  User ID whose sessions should be revoked
</ParamField>

#### Response

<ResponseField name="void" type="void">
  Returns void on success
</ResponseField>

<Warning>
  You **cannot revoke all of your own sessions**. This prevents accidental lockout.
</Warning>

#### Code Example

<CodeGroup>

```typescript React Query
import { orpc } from '@/lib/orpc/client';

function RevokeAllSessionsButton({ userId, currentUserId }) {
  const { mutate, isPending } = orpc.user.revokeUserSessions.useMutation({
    onSuccess: () => {
      orpc.user.getUserSessions.invalidate({ input: { userId } });
      alert('All sessions revoked successfully');
    },
  });

  const handleRevoke = () => {
    if (userId === currentUserId) {
      alert('You cannot revoke all of your own sessions');
      return;
    }
    
    if (confirm('Revoke all sessions? The user will be logged out from all devices.')) {
      mutate({ input: { id: userId } });
    }
  };

  return (
    <button 
      onClick={handleRevoke} 
      disabled={isPending || userId === currentUserId}
    >
      Revoke All Sessions
    </button>
  );
}
```

```typescript React Query
import { orpc } from '@/lib/orpc/client';

function RevokeSessionButton({ userId, sessionToken, currentSessionToken }) {
  const { mutate, isPending } = orpc.user.revokeUserSession.useMutation({
    onSuccess: () => {
      orpc.user.getUserSessions.invalidate({ input: { userId } });
    },
  });

  const handleRevoke = () => {
    if (sessionToken === currentSessionToken) {
      alert('You cannot revoke your current session');
      return;
    }
    
    mutate({ input: { id: userId, sessionToken } });
  };

  const isCurrentSession = sessionToken === currentSessionToken;

  return (
    <button 
      onClick={handleRevoke} 
      disabled={isPending || isCurrentSession}
      title={isCurrentSession ? 'Current session' : 'Revoke this session'}
    >
      {isCurrentSession ? 'Current' : 'Revoke'}
    </button>
  );
}
```

## Error Handling

<ResponseField name="UNAUTHORIZED" type="401">
  Missing or invalid authentication credentials
</ResponseField>

<ResponseField name="FORBIDDEN" type="403">
  Insufficient permissions for the operation
</ResponseField>

<ResponseField name="NOT_FOUND" type="404">
  User with the specified ID does not exist
</ResponseField>

<ResponseField name="CONFLICT" type="409">
  Email already exists when creating/updating user
</ResponseField>

<ResponseField name="BAD_REQUEST" type="400">
  - Attempting to delete yourself
  - Attempting to revoke your own sessions
  - Invalid input parameters
</ResponseField>
